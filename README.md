# Hey, I'm Aryan 👋

I build AI systems for physical infrastructure — data centres, power grids, supply chains.

Currently working on **Veltora**, a predictive intelligence platform for data centre operators. The core idea: give infrastructure teams the kind of situational awareness that's standard in aerospace and finance, but basically nonexistent in the physical compute layer.

Before this I spent time at the intersection of applied ML and systems engineering — mostly time-series forecasting, multi-agent RL, and real-time telemetry pipelines. Got obsessed with data centres when I realised the AI boom is fundamentally bottlenecked by physical infrastructure, and nobody has really solved the intelligence layer there yet.
